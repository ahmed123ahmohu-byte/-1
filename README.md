# Nexa

واجهة تطبيق Flutter احترافية لـ **Nexa** بهوية مستقبلية عربية:

- Splash Screen مع Typewriter + Glow/Particles.
- شاشة دخول بسيطة جدًا (اسم فقط).
- شات حديث بستايل Dark/Glassmorphism.
- Typing Indicator متحرك.
- وضع أكواد يقسم الشاشة تلقائيًا (Code Preview + Chat).

## تشغيل سريع

```bash
flutter pub get
flutter run
```

## ملف YAML للهاتف

تم إعداد `pubspec.yaml` ليعمل مباشرة كتطبيق Flutter للهاتف (Android/iOS).
